# Lido Multi-Semantic Release

A wrapper around [semantic-release](https://github.com/semantic-release/semantic-release) that makes it work with monorepos.

## Installation

```bash
yarn install
yarn build
```

## Usage

```bash
multi-semantic-release
```

## Development

```bash
# Install dependencies
yarn install

# Development mode with hot reload
yarn dev

# Build for production
yarn build

# Lint code with ESLint
yarn lint

# Format code with Prettier
yarn format

# Run TypeScript type checking
yarn typecheck
```
